# 快速开始

欢迎来到 Claude Code 完全指南！本指南将帮助你快速上手 Claude Code。

## 什么是 Claude Code？

Claude Code 是 Anthropic 官方推出的 AI 编程助手，可以直接在你的终端中使用。它不仅能理解你的代码库，还能：

- 🚀 从自然语言描述直接生成功能
- 🐛 智能分析和修复 Bug
- 🗺️ 回答关于代码库的任何问题
- ⚡ 自动化繁琐的开发任务

## 安装

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

## 首次使用

安装完成后，运行以下命令启动 Claude Code：

```bash
claude
```

首次使用时会提示你登录 Claude.ai 账户。

## 你的第一个命令

**用户**：
```
解释这个项目是做什么的？
```

**Claude Code**：
```
我会分析项目结构，了解代码库的组织方式、主要功能和技术栈。
让我看看项目中的关键文件...
```
